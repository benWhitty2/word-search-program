This program automatically reads and solves images of word searches using k nearest neighbour to classify the individual letters and then using a closest matching system 
that uses a confusions table created in the training process to predict the likelihood of one letter being mistaken for another.

To run the program
run the command:  python code/evaluate.py
from the word-search-program directory
to get an image of the solved word search 
run: python code/evaluate.py --display
