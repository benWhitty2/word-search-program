This program automatically reads and solves images of word searches using k nearest neighbour to classify the individual letters. It them uses a closest matching system 
that uses a confusions table created in the training process to predict the likelihood of one letter being mistaken for another. In order to find the most likely set of letters of the same length that is the correct word, the position of this word is then recorded and checked against the answers at the end of the program.

# To run the program

from the word-search-program directory

run the command:  python code/evaluate.py

to get an image of the solved word search 

run: python code/evaluate.py --display
